## 🔍 查找模块

### 搜索技巧

1. **按流行度排序**: 默认按"Most installed"排序
2. **按版本筛选**: 选择"Works with" Drupal 版本
3. **按类别筛选**: 使用左侧类别过滤器
4. **关键词搜索**: 在搜索框输入功能关键词
5. **维护状态**: 筛选"Maintained"模块

### 评估模块

| 指标 | 说明 |
|------|------|
| **安装量** | 越高表示越受欢迎 |
| **维护状态** | 选择"Actively maintained" |
| **安全覆盖** | 确保有安全团队覆盖 |
| **更新频率** | 查看最近更新时间 |
| **问题队列** | 检查未解决问题数量 |
| **文档质量** | 查看是否有完整文档 |
